Save the map file as "CollisionMapGenerator-android/assets/data/TileMap(base64).xml"
	make sure it's base64 encoded, with no compression

Collision file is saved as "CollisionMapGenerator-desktop/Collisions.physicsmap"


Editting:
	WASD to move camera
	use mouse wheel to zoom
	left click to set tile
	right click to unset tile
	click and drag to set/unset multiple tiles
	Enter to export file
